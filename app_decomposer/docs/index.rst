Welcome to app_decomposer's documentation!
======================================
**App Decomposer** is a python package that extract data and timeseries from an application instrumented with IO-Instrumentation on a cluster. The job data flow profile and volume is then analyzed and decomposed into compute-dominant and I/O-dominant phases. Result of the decomposition is presented in multiarray form encording the seuqnetial behaviour of the application and can be used to feed the Execution Simulator to produce a simulation.
